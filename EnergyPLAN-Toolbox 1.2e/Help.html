EnergyPLAN calling execution from Matlab

This document show how to use energyPlan Matlab function to call EnergyPLAN Advanced Energy System Analysis Computer Model [1].

System Requirements

To use this toolbox you need the following requirements:

  • Requires MATLAB R2014b or later.
  • Requires EnergyPLAN.
  • Requires EnergyPLAN Toolbox

Paths and Folder definitions

To begin to work with this function you have to specify:

  1. The EnergyPLAN path, that it is the path address where it is located in your computer the .exe file of EnergyPLAN advanced energy system analysis computer model.
  2. The input file path, that it is the path address where it is located in your computer your reference model input that you want to use in EnergyPLAN.
  3. The ouput folder where you want to save your results.
%Path where it is located EnergyPLAN executable.
energyPlanPath = 'C:\Users\Usuario\Desktop\ZIPEnergyPLAN142\ZIPEnergyPLAN142\energyPLAN.exe';
%Path of reference file.
inputFilePath = 'C:\Users\Usuario\Desktop\ZIPEnergyPLAN142\ZIPEnergyPLAN142\energyPlan Data\Data\Denmark2030Alternative.txt';
%Folder where will be output resulf of EnergyPLAN.
outputFolder = 'C:\Users\Usuario\Desktop\ZIPEnergyPLAN142\ZIPEnergyPLAN142\Outputs\';

EnergyPLAN calling and parameters/values definitions.

After definitions of the principal sources of the function you can define some values that you want to change in your reference model execution.

Matlab doesn't change your reference model, it just reads your reference model, generates a new .text input file with the changes that you want to do and then executes this file to save results where you indicate.

In this example It is only showing how to change the Onshore and Offshore Wind Installed Capacities of Denmark2030Alternative.text reference model. Also, It is indicated the regulation strategy that EnergyPLAN has to employ. In this way it is possible to see how to specify whatever input parameter of EnergyPLAN input file. You only will need to know the name of parameter located in your reference input file, introduce this name followed his numeric or string value in the input part of energyPlan function execute in Matlab.

% Definition of numeric values to parameters of reference input file.
OnshoreWind=3500;
OffshoreWind=3100;
STR='23457000';

%EnergyPLAN calling from matlab.
[annualData, monthlyData, hourlyData, PES, RESshare, TotalAnnualCosts, VariableCosts]=energyPlan(energyPlanPath, inputFilePath, outputFolder,...
    'input_RES1_capacity=', OnshoreWind, 'input_RES2_capacity=', OffshoreWind,...
    'input_keol_reg=', STR);
